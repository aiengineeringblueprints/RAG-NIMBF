# Claude Instructions

This repository has an Obsidian vault at `obsidian-vault/`.

Before broad project exploration, consult the vault first:

- Start with `obsidian-vault/00-Index/Home.md`.
- Follow links to the relevant architecture, module, operations, testing, and research notes.
- Use source files to verify details when implementing or reviewing changes.

When code behavior changes, update the vault in the same change:

- Architecture or flow changes: update `obsidian-vault/01-Architecture/`.
- Module/API/config changes: update `obsidian-vault/02-Modules/`.
- Run commands, environment variables, outputs, or services: update `obsidian-vault/03-Operations/Operations Runbook.md` and `obsidian-vault/02-Modules/Configuration Reference.md`.
- Research/result interpretation changes: update `obsidian-vault/04-Research/Research Notes.md`.
- Test changes: update `obsidian-vault/05-Testing/Testing and Coverage.md`.

If vault notes and source disagree, treat the source as authoritative, then correct the vault.
